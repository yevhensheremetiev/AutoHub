# AutoHub

Monorepo for AutoHub project.

## Stack

- Backend: NestJS, PostgreSQL, Prisma, Firebase (Google auth), `@nestjs/config`
- Frontend: Vite, React + TypeScript, React Router, TanStack Query, React Hook Form + Zod, Tailwind CSS, shadcn/ui

## Structure

- `apps/backend/` — NestJS API
- `apps/frontend/` — React client
- `packages/shared/` — shared types/utils
